        4.1软件功能：本软件提供手机体检、手机杀毒、系统清理加速、隐私加密保护、拦截骚扰信息、流量监控、节电管理、软件管理、安全备份、手机防盗以及文件管理、快速拨号等系统管理功能。

            4.1.1本软件包含名为“联网云查杀”的杀毒扫描功能。为了防止病毒木马及恶意软件入侵用户的手机系统，如果用户已确认开启自动联网云查杀，将在杀毒扫描之后，自动访问奇虎360手机云安全中心进行联网云查杀，并根据软件特征信息（软件的UID、包名、名称、版本号、开发厂商、证书），判定手机中已安装软件及软件安装包的安全级别，提示用户进行相应处理。联网云查杀需要经过用户的确认才会执行，用户也可以通过关闭“自动联网云查杀”设置项来关闭此功能。

            4.1.2安全备份：用户使用“安全备份”功能，必须首先进行帐户注册与激活。只有在用户登陆，并由用户主动选择进行备份通讯录或恢复通讯录的操作的情况下，安全备份功能才会启动。该功能会将用户手机中的通讯录备份到奇虎360公司的服务器或从奇虎360公司的服务器恢复到本地。无论备份或恢复过程，都通过加密方式进行传输。用户登陆后，还可以查看近10次的备份记录。

            4.1.3为了实现拦截垃圾短信，拦截骚扰电话的功能，本软件中包含名为“骚扰拦截”的功能模块。用户可以选择号码加入黑名单和白名单，并选择需要的拦截模式，达到期望的拦截效果。拦截模式包含标准模式、只拦截黑名单、只接受通讯录、只接受白名单、全部拦截以及自定模式，根据用户所选的模式，将对黑名单、白名单、联系人短信、联系人电话、陌生人短信、陌生人电话和隐藏号码电话采用对应的处理方式。在判断来电和短信是否需要拦截时，可能会读取用户手机中的联系人名单进行比对；为了智能识别过滤垃圾短信，需要读取用户手机中的通话记录，并对陌生号码的短信内容进行特征分析，若短信内容中包含垃圾短信特征，则对该短信进行拦截。

            4.1.4流量监控：为保证流量统计的实时性与准确性，会通过手机系统接口检测流量使用量并记录，并且对这些记录信息进行一定分类和计算，最终以最清晰直观的方式展现给您。 在您的主动选择情况下，360手机卫士会对某个软件的联网行为进行禁用，目的是为您进一步提供节约流量的途径或者警示流量使用不正常的软件。

            4.1.5手机防盗： 手机防盗功能通过换卡短信通知、远程删除数据、追踪手机位置、遥控响警报音、远程锁定。使用手机防盗功能，当检测到SIM卡发生改变时，您的手机会自动向您预先设置的安全号码发送通知短信。您可以凭此短信获取手机当前的手机号码。您可以通过发送防盗指令对手机执行远程数据删除、追踪手机位置和锁定手机的操作。该动作需要您的预先设置和手动触发，并对您存储在手机中的数据造成永久性损坏，请谨慎操作。

            4.1.6密码保险箱：密码保险箱提供用户账号密码信息的本地加密保存，不会对用户的设备和系统造成任何改变和影响，也没有任何的数据上传。密码保险箱采用国际标准加密算法对您的数据进行加密保护。卸载密码保险箱用户加密保存的数据也会被清除。用户设置的进入密码保险箱的口令是进入软件的唯一凭证，请牢记。口令忘记，将无法进入密码保险箱。

            4.1.8 360 宝盒：360宝盒能为您提供安全快速地下载应用；方便地安装软件，轻松地管理手机上的软件，包括卸载，升级等。360宝盒在启动后，会显示您的手机软件中哪些可以进行升级，此时需要读取并联网检测用户手机中的安装的软件版本、软件UID、IMEI、机型信息、系统版本、屏幕分辨率信息。您可以通过“设置”功能，将“升级检测”的选项设为“关闭”，之后360宝盒将不会检测以上任何信息。当用户在360宝盒中对其中的软件发表评论时，360宝盒会在评论内容中显示出用户的机型（塞班同一款软件包含多个版本，从提高用户体验考虑，针对机型的评论可以让用户得到针对自己手机更有参考价值的信息）、系统信息，供有相同机型和系统的其他用户进行参考，因此需要读取并显示您手机的机型和系统版本信息。

            4.1.9 360优化大师能够对手机系统进行一键加速，完成包括进程清理、缓存垃圾清理、开机启动项清理和系统垃圾深度清理。节电管理能够显示手机电池状态信息并能够通过快捷开关系统耗电功能开关项，达到有效延长电池使用时间的目的。软件管理能够方便对手机中的应用进行安装、删除、备份和移动至SD卡等操作。文件管理可以对手机中的文件进行管理，可以按传统列表模式和文件类型分类浏览，能够对文件进行打开、复制、粘贴、重命名等操作，支持全盘查找指定文件。快捷设置可以对系统常用功能开关进行一站式管理，包括铃声调节、一键设置上网接入点、来电/短信铃声设置等。
